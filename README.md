# GARÇOM APP #

O aplicativo Garçom App tem como objetivo auxiliar o garçom
de um restaurante a administrar os pedidos dos clientes por
mesa.

## **Primeiro Aplicativo** ##

Este foi o primeiro aplicativo que desenvolvi, como trabalho para a disciplina Técnicas de Programação, no fim do segundo período de 2014 do curso de graduação. O aplicativo foi desenvolvido em aproximadamente duas semanas.
